Welcome to the Collection Registry for the UC Libraries Digital Collection (UCLDC)!

The Collection Registry is an interface for staff at the UC campus libraries to define, track, and manage information about collections in the UCLDC. It is a key component of the UCLDC that will serve three key purposes. It will:

  1. Drive some essential display requirements for the UCLDC public interface, for example faceting search results by campus name or collection.
  2. Enable campus libraries to denote and track the status of digital collections slated for the UCLDC, for example by indicating the location of collections ready for harvest.
  3. Provide a platform for collaborative digital collection development and/or digitization planning.

This interface is under development, as part of the broader implementation of the UCLDC. Please report any bugs or feedback to ucldc@ucop.edu.

For more information about the UCLDC and the Implementation Project, visit our project wiki.
